To run the project open the file group_20_Assignment7.pde and then click the run button to run the program.

Use the arrow keys to move right, left, to jump, or to slide. Click on the pause button to pause the game and the play button to resume play. The home button will quit the game and the circular arrow button will restart the game.

Goal of the game: collect as many coins as you can without dying from the zombie or water and make it to the victory flag. Bonus points are awarded for not losing lives.
